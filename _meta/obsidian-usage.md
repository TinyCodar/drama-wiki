# Obsidian usage

Open `/Users/tinycoder/wiki` as a vault in Obsidian.

Recommended views:
- Graph View: inspect wiki links
- Local Graph: inspect one topic neighborhood
- Search: find concepts/entities

Current vault-specific settings written to `.obsidian/`:
- attachment folder: `raw/assets`
- graph shows tags and attachments
- core graph/backlink/search plugins enabled
